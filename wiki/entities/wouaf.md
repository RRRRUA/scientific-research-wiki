# WOUAF

WOUAF is a user-attribution method for text-to-image diffusion models introduced by [[kim-2024-wouaf]]. It modulates decoder weights with a user fingerprint so each user receives a distinct traceable model copy.

In this project, WOUAF represents the transition from a fixed watermark to distributor-centered user attribution.
