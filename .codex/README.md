# Codex Project Configuration

This folder replaces the `.kiro/agents` convention used by the reference repository.

- `agents/`: role prompts adapted for Codex-style use.
- `skills/research-wiki/SKILL.md`: project-local workflow skill for this research wiki.
- `skills/research-wiki-publisher/SKILL.md`: validate, commit, and push timestamped git snapshots.

The executable tools live in `tools/wiki/` at the repository root.
